# gChess Frontend

Angular 21 web chess client for the gChess platform.

## Prerequisites

- Node.js + npm
- A running [gChess backend](../gChess-back) on `http://localhost:8080`

## Development server

```bash
npm install
npm start
```

Open your browser at `http://localhost:4200/`. The application reloads automatically on file changes.

## Building

```bash
npm run build
```

Artifacts are output to the `dist/` directory. The production build is optimized for performance.

## Running unit tests

```bash
npm test
```

## Code scaffolding

Angular CLI is available via npx:

```bash
npx ng generate component component-name
npx ng generate --help
```
